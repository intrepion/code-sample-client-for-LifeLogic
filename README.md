# code-sample-client-for-lifelogic

## Commands

### Install

```bash
npm install
```

### Test

```bash
npm test
```

### Initialize Database

```bash
./scripts/init_postgres.sh
```

### Run

```bash
REACT_APP_SERVER_URL="http://localhost:5147" npm start
```
